# URL Shortener CLI (Java)

Консольный сокращатель ссылок: уникальные короткие ссылки на пользователя, лимиты переходов, срок жизни, уведомления, открытие в браузере.

## Возможности
- Сокращение URL → короткая ссылка вида `clck.ru/AAAA`
- Уникальные ссылки для каждого пользователя (UUID)
- Лимит переходов и блокировка при исчерпании
- Время жизни (TTL) задаётся системой (по умолчанию — 24 часа), автоудаление «протухших» ссылок
- Уведомления пользователю об истечении/лимите
- Переход по короткой ссылке открывает браузер

## Требования
- Java 17+
- Maven 3.8+
